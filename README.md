# Plugins and Extensions for CS:GO Zombie Escape

This repository was created to help CS:GO server administrators who run the Zombie Escape mode.  
It contains a collection of plugins and configuration files that make it easier to set up a server and keep it stable.

It is important to note that I am not the author of most of these plugins.  
They were originally developed by other community members, and their original versions are publicly available.  
My role was limited to:
- adapting some plugins for SourceMod 1.12;
- fixing compilation errors;
- making minor adjustments for stable operation.

In short, this project is not a new development.  
It is a collected package designed to help other administrators quickly set up a Zombie Escape server and avoid common issues during compilation and startup.

**Note:** This is not a full modification and is not a complete build of Zombie Escape.
